# crypto_bot
Репозиторий для обновления скрипта по криптоботу: json файл с данными для пуша на сайт,
текстовый файл, отдельный файл с функциями создания текста и пуша, основной код в main_2.py
Код запускается главной функцией в main_2.py, обращение к API, составление текста и пуш
Используемые API: 

CoinMarketCap API
Binance API
CoinBase API
OKEX API

Python 3.11.4
